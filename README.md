## Practica 2.1

Autor: Mateo

Código fuente de la Tienda online


## 1. Construir la imagen

Desde la raíz del proyecto:

```bash
docker build -t tienda-online .
```

## 2. Ejecutar la imagen

Ejemplo:

```bash
docker run --rm tienda-online
```

## 3. Salida esperada

Al ejecutar el contenedor se mostrará el menú principal de la aplicación.
